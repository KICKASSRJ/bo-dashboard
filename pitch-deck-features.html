Dashboard Landing Page

A clean, card-based home screen that gives users one-click access to every capability. No menus, no training — just click what you need.

  • 5 feature cards with clear descriptions

Feature 1
📁

SAP Input Data

Upload 4 SAP Excel exports — EDIDC, MSEG, EKES, and RSN — individually or all at once with auto-detection by filename.

  • Bulk upload with automatic file-type detection
  • Row counts shown per file (e.g. 7,368 rows)
  • Background parsing via Web Worker — no page freezes
  • DRM-protected file detection with guidance
  • Replace or remove files at any time
  • Data persists across page refreshes (IndexedDB)
Feature 2
📋

BOR & GR Message Status

Displays all IDoc records from EDIDC with full status descriptions, sortable columns, and a real-time text filter.

  • Full table of 1,000+ IDoc records at a glance
  • Click any column header to sort ascending/descending
  • Filter by Message Type, IDoc Number, Status, or any field
  • Human-readable status descriptions (e.g. "Application document posted")
  • EDI Archive Key visible for CID cross-reference
Feature 3
🔎

CID Processing Status

Look up any Correlation ID (EDI Archive Key) to instantly find its IDoc number, processing status, message type, and creation date.

  • Paste a CID and hit Search — results in milliseconds
  • Sample CIDs from uploaded data shown as clickable buttons
  • Shows all matching records for the same CID
  • Color-coded result rows for quick scanning
Input — Sample CIDs from uploaded data Output — Search result
Feature 4
✅

RSN Status Verification

Verify whether Return Service Notification numbers have been successfully received into SAP ECC — one at a time or in batch.

  • Enter RSNs comma-separated or one per line
  • Sample RSN values shown as clickable buttons
  • Clear "found in SAP ECC" confirmation badge
  • Batch check multiple RSNs simultaneously
Input — Sample RSNs from uploaded data Output — Verification result
Feature 5
⚠️

BOR / GR Mismatch Detection

Automatically compares every EKES (BOR) record against MSEG (Goods Receipt) data and flags mismatches — no manual input required.

  • All 7,000+ records compared automatically on load
  • Green rows = "BOR and GR are in sync"
  • Red rows = "BOR FG and GR Mismatch"
  • Filter buttons: All / Mismatches / In Sync
  • Search by BO number or PID
  • Material Document column from MSEG for traceability
